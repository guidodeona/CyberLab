# CyberLab

CyberLab es una herramienta web de ciberseguridad desarrollada con Next.js y TypeScript.

## Características

* Análisis de URLs
* Consulta mediante API propia
* Visualización de resultados
* Base para análisis de headers de seguridad

## Tecnologías utilizadas

* Next.js
* React
* TypeScript
* Tailwind CSS

## Objetivo

Crear una plataforma que permita analizar configuraciones de seguridad web y generar recomendaciones para mejorar la postura de seguridad de un sitio.
