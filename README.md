# CyberLab 🛡️

CyberLab es una plataforma de análisis de seguridad web desarrollada con Next.js, TypeScript y Tailwind CSS.

Permite realizar evaluaciones rápidas de la postura de seguridad de sitios web, generando informes detallados y reportes PDF profesionales.

---

## Características

### Seguridad Web

* Análisis SSL/TLS
* Detección de HTTPS
* Evaluación de Security Headers
* Cálculo de Security Score
* Clasificación de Nivel de Riesgo

### Inteligencia de Infraestructura

* DNS Lookup
* WHOIS / RDAP
* Geolocalización IP
* Información del ISP
* Tiempo de respuesta HTTP

### Detección Tecnológica

* CMS Detection
* WAF Detection
* Identificación de tecnologías web

### Reportes

* Exportación PDF profesional
* Resumen ejecutivo
* Recomendaciones de seguridad
* Historial de análisis

### Interfaz

* Dashboard visual
* Modo Oscuro / Claro
* Diseño responsive
* Experiencia optimizada para escritorio y móvil

## Instalación

Clonar repositorio:

```bash
git clone https://github.com/TU-USUARIO/CyberLab.git
```

Ingresar al proyecto:

```bash
cd CyberLab
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en modo desarrollo:

```bash
npm run dev
```

Abrir:

```text
http://localhost:3000
```
